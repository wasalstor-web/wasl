للمطورين: تحتاج هذه الصفحة إلى مفتاح API من OpenWeatherMap للعمل بشكل كامل.

1. احصل على مفتاح API مجاني من OpenWeatherMap

2. أضف المفتاح كمتغير بيئي في ملف .env.local:

NEXT_PUBLIC_OPENWEATHER_API_KEY=your_api_key_here

3. في الوقت الحالي، يتم استخدام مفتاح تجريبي محدود لأغراض العرض.
